Programming Languages
Machine language Sequences of instructions that are encoded as numeric digits.
Debugging A tedious process of locating and correcting errors within programs.
Assembler Programs developed to convert mnemonic expressions into machine language instructions.
Assembly language A mnemonic system for representing programs.
Translator A program that translates expressed in these high-level primitives into machine-language programs.
Compiler A program that translates a programming language’s source code into machine code, bytecode or another programming language.
Interpreter A program that executes the instructions as they were translated instead of recording the translated version for future use.
Cross-platform software Software that is independent of an operating system’s design as well as the machine’s hardware design and is therefore executable throughout a network.
Natural language Forms of communication that evolve over time without formal grammatical analysis.
Formal language Forms of communication that are precisely defined by grammars.
Machine independence A programming language that does not rely on the characteristics of a particular machine.
High-level programming language A computer programming language that isn't limited by the computer, designed for a specific job, and is easier to understand.
Object-oriented programming (OOP) A computer programming model that organizes software design around data, or objects, rather than functions and logic.
Scripting language Programs used to perform administrative tasks rather than to develop complex programs.
Data Types
  1. Integer
  2. Float
  3. String
  1. Numeric data consisting of whole numbers, probably stored using two’s complement notation.
  2. Numeric data that might contain values other than whole numbers, probably stored in floating-point notation.
  3. A representation of textual data that is encoded as a sequence of 16-bit unsigned integer values representing UTF-16 code units.
Primitive data type The data types that are included as primitives in a programming language.
Data structure The conceptual shape or arrangement of data.
Python collection data structures
  1. List
  2. Tuple
  3. Set
  4. Dictionary
  1. A collection which is ordered and changeable. Allows duplicate members.
  2. A collection which is ordered and unchangeable. Allows duplicate members.
  3. A collection which is unordered, unchangeable, and unindexed. No duplicate members.
  4. A collection which is ordered (Ver. 3.7) and changeable. No duplicate members.
Function A set of instructions for performing a task that can be used as an abstract tool by other program units.
Local variable A variable declared within a function that can only be referenced within that function.
Global variable Variables whose scopes are not restricted to a particular part of a program.
Source program A program in its original form.
Object program The translated version of a program.
